# Cooking Masterclass Catalogue

## Overview
A Vue.js single-page app showcasing cooking courses with wishlist functionality. 

## Features
- Course cards with title, chef, level, price
- Wishlist counter in header
- "Sold Out" indication
- Responsive design

## Installation
```bash 
git clone https://github.com/lukhangelelandu-crypto/cooking-masterclass-catalogue.git
cd cooking-masterclass-catalogue
npm install
npm run dev
